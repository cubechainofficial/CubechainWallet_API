# 변수 형식 설명 필수 비고
1 fromWallet String 토큰 생성 주소 * 변경되지 않음
2 tokenSymbol String 토큰 심볼 * 변경되지 않음
3 tokenName String 토큰명
4 totalAmount Number 토큰 총 발행량
5 tokenDecimal Number 토큰 소수점 자리수 Default : 8
6 description String 토큰 소개
7 officialUrl String 발행처 Url
8 socialLink String 발행처 소셜미디어
9 iconUrl String 아이콘 Url
10 setBurn Number 소각 기능 1: 전체소각, 2: 개별소각, 0: 사용안함(Default)
11 setLock Number 락 기능 1: 전체락, 2: 개별락, 0: 사용안함(Default)
12 setMint Boolean 추가발행 기능 여부 Default : false
13 feeWallet String 수수료 지불 지갑 설정시 유저는 지불하지 않고 설정된 지갑에서 수수료가 차감됨
15 privateKey String 생성 지갑주소의 개인키 *

response : result.data
